React Weather Component

  1. A resuable react ui weather Component
    2. It has 2 props out of which 1 is required (mandatory)
  2. apiKey - weather api (mandatory prop)
  3. BG - background of the Component. Can be anything like image url, color gradien, hsl, rgb/rgba or any color (not mandatory)

Usage

  • import the component
    • import WeatherComponent from '../Components/weathercomponent'
  • Use the UI component
    • <WeatherComponent apiKey="b0b80ecb1ac34990a46201357242012"'/> <WeatherComponent apiKey="b0b80ecb1ac34990a46201357242012" BG='linear-gradient(135deg, #6e8efb, #a777e3)'/>

Command-Line Program

Write a basic Node.js program that accepts command-line arguments and processes them.
Level 1

Problem Description: 1. Create a script that calculates the sum of numbers passed as command-line arguments. 2. Use process.argv to parse and handle the arguments. 3. Include error handling for invalid inputs (e.g., non-numeric values).

Level 2

Problem Description: 1. Enhance the script to accept a --multiply flag to multiply the numbers instead of adding them. 2. Update the help text to include usage instructions.

Level 3

Problem Description: 1. Modularize the program by creating a separate file (mathutils.js) for the add and multiply functions. 2. Now, take the operation followed by operands as the arguments. Like command add 1 2 or command multiply 3 4. 3. Create a script using bin part of the package.json. Use npm link to test the script

Debugging (debug)

Use the debug package for selective debugging in multi-module scenarios.
Level 1

Problem Description: 1. Install debug and set up a namespace for debugging. 2. Use the debug function to log messages in a single module.

Level 2

Problem Description: 1. Extend the program with multiple modules, each having its own debug namespace. 2. Use environment variables to selectively enable debugging for specific modules.

Level 3

Problem Description: 1. Combine debug with winston for structured debugging and logging. 2. Use both tools to debug and log a complex program scenario. For the complex program, Enhance the compute program by making it interactive program.

Setting Up npm

Learn to configure and use npm for managing scripts, dependencies, and modules effectively.
Level 1

Problem Description: 1. Initialize a new Node.js project using npm init. 2. Add a start script to run a basic index.js file. 3. Add a dev script to run the same file with debugging output (DEBUG=* node index.js).

Level 2

Problem Description: 1. Install the chalk package and use it in index.js to style console outputs. 2. Add nodemon as a dev dependency and use it to automatically restart the program during development.

Level 3

Problem Description: 1. Add a prestart script to ensure node_modules is available before running the program. 2. Test by removing node_modules and ensuring npm start runs successfully after reinstalling dependencies.
